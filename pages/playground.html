Classes

  • ex:Shrine — Ise Jingu, Izumo Taisha, Fushimi Inari, Meiji Jingu, Kasuga Taisha
  • ex:Deity — Amaterasu, Okuninushi, Inari, Takemikazuchi
  • ex:Person — Tanaka, Suzuki (shrine priests)
  • ex:Myth — Kojiki, Nihon Shoki

Properties

  • ex:name, ex:alternateName — text labels
  • ex:foundedYear — integer (e.g. 711)
  • ex:rank — integer 1-5
  • ex:locatedIn — Mie, Shimane, Kyoto, Tokyo, Nara, Osaka
  • ex:enshrines — shrine → deity
  • ex:domain — deity → Sun/Earth/Harvest/Thunder
  • ex:appearsIn — deity → myth
  • ex:servesAt — person → shrine (temporal!)
  • ex:hasEmbedding — 4D vector (HNSW indexed)

RDF-star Edges

  • << shrine enshrines deity >> ex:confidence — "0.99", "0.95", "0.70"
  • << shrine enshrines deity >> ex:source — provenance string

Temporal Annotations

  • Tanaka servesAt Ise: valid [1900, 1950]
  • Suzuki servesAt Ise: valid [1950, ∞)
  • Fushimi locatedIn Osaka: valid [1467, 1499] (Onin War)
  • Fushimi locatedIn Kyoto: valid [711, ∞)
  • Meiji locatedIn Tokyo: valid [1920, ∞)
