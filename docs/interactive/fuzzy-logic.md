# Fuzzy logic explorer

Pick a connective from the tabs. Move the sliders for `a` and `b`. Every output is a smooth polynomial (or a single product, for XOR and IFF) — no branches, no sign function, no kinks. Because the operator is a pure tensor computation, the program that uses it runs as one chain of kernel launches with no runtime control flow, and gradients flow through end-to-end.
